2. Персональные данные, подлежащие обработке

2.1. Оператор может обрабатывать следующие персональные данные Пользователя:

  • Фамилия, имя, отчество;
  • Адрес электронной почты (email);
  • Имя пользователя в Telegram;
  • Данные об оплате (без хранения данных банковских карт — обработка осуществляется платёжным провайдером);
  • Техническая информация (IP-адрес, тип браузера, время посещения).

3. Цели обработки персональных данных

3.1. Персональные данные обрабатываются в следующих целях:

  • Предоставление доступа к Продукту «Niuma»;
  • Обработка платежей и выставление чеков;
  • Оказание технической поддержки;
  • Информирование об обновлениях Продукта;
  • Выполнение обязательств, предусмотренных Договором оферты.

4. Правовые основания обработки

4.1. Обработка персональных данных осуществляется на следующих правовых основаниях:

  • Согласие субъекта персональных данных;
  • Исполнение договора, стороной которого является субъект персональных данных;
  • Соблюдение требований законодательства Российской Федерации.

5. Порядок обработки персональных данных

5.1. Оператор обеспечивает сохранность персональных данных и принимает все возможные меры, исключающие доступ к персональным данным неуполномоченных лиц.

5.2. Персональные данные Пользователя никогда, ни при каких условиях не будут переданы третьим лицам, за исключением случаев, связанных с исполнением действующего законодательства, а также случаев, когда передача данных необходима для обработки платежей (платёжному провайдеру).

5.3. Программный продукт «Niuma» работает локально на компьютере Пользователя. Код, файлы, project rules и содержимое диалогов из IDE не передаются на серверы Оператора. Серверные запросы используются только для покупки, проверки лицензии, получения обновлений и сопутствующих сервисных операций.

6.1. Персональные данные обрабатываются в течение срока действия подписки и в течение 3 (трёх) лет после её окончания в целях бухгалтерского учёта.

6.2. По истечении срока обработки персональные данные уничтожаются.

  • Получить информацию об обработке своих персональных данных;

  • Отозвать согласие на обработку персональных данных, направив запрос на email: support@nc43.ru.
